# Quick Start Guide:

1. Add link to package in head
2. Add the 'class="placeholdify"' to an element or body to turn entire page into wireframe

```html
<html>
  <head>
    <link
      href="https://unpkg.com/placeholdifier/placeholdifier.css"
      rel="stylesheet"
    />
  </head>
  <body class="placeholdify">
    <h1>Hello World</h1>
    <h1>Hello World</h1>
  </body>
</html>
```
